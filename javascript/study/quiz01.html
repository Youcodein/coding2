자바스크립트 사용 방법

자바스크립트의 사용방법은
1. 외부파일 연결하는 방법
2. 스크립트 태그에 사용하는 방법
3. 태그에 직접 기술하는 방법 등이 있습니다. 

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Javasript</title>
    <!-- 1. 외부 파일로 로드하는 방법 -->
    <script type="text/Javasript" src="js/script.js"></script>
    <!-- 2. script 태그 사이에 기술하는 방법 -->
    <script>
        document.write("Javasript");
    </script>
</head>
<body>
    <!-- 3. 태그에 직접 기술하는 방법 -->
    <a href="#" onclick="alert('Javasript')"></a>
</body>
</html>
